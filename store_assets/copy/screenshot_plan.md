## Especificações técnicas

| Atributo | Valor |
|---|---|
| Formato | JPEG ou PNG (sem transparência) |
| Dimensão mínima | 320px em qualquer lado |
| Dimensão máxima | 3840px em qualquer lado |
| Resolução recomendada | 1080×1920px (portrait, 9:16) |
| Orientação | Portrait |
| Conteúdo | Apenas UI do próprio app — sem elementos de jogos terceiros |

## Ordem na ficha — raciocínio

A sequência conta uma história progressiva:

```
1. Impacto visual     → gancho emocional, primeira impressão
2. Funcionalidade     → o que o app realmente faz
3. Personalização     → profundidade por jogo
4. Organização        → biblioteca de jogos
5. Detalhamento       → GFX Profile
6. Comprometimento    → histórico de sessões
7. Visão geral        → dashboard Home
8. Confiança          → configurações, política, idioma
```
