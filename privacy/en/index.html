Privacy Policy — Apex Booster+

Português | English | Español

App name: Apex Booster+

Developer / Controller: AllAppsEngineer

Official contact: alessandro.lopes.adm@gmail.com

2. Locally Stored Data

Apex Booster+ does not have its own server and does not send personal data to external servers. All the data below is stored exclusively on the user's device, through Android's SharedPreferences:

DataPurpose
Library of added gamesDisplay and manage the user's game library
Game identifier (packageName)Identify and launch the corresponding app
Local GFX profile per gameSave the visual preference chosen by the user
Session historyDisplay previous sessions and local metrics
Selected languageKeep the language preference between sessions
Onboarding completed flagAvoid showing the tutorial again when reopening the app

This data is not synced, sold, or shared with third parties.

3. Local Device Readings

The app performs the following local readings for informational purposes:

  • Available and total RAM: reads the device's memory to display the session snapshot.
  • Basic network latency (Apex Ping): the app's own connectivity test to assess connection quality during the session. It does not represent or promise the actual ping of external game servers.
  • Focus Mode / Do Not Disturb (DND) status: checked only when the user grants notification policy access permission. It is not enabled automatically.

These readings are never sent to any server.

4. Internet Use

The app uses the internet permission exclusively for the basic latency and connectivity test mentioned above (Apex Ping). No other network communication is performed. The app does not collect, transmit, or store data on external servers.

5. Installed Apps

To allow the user to select games from their library of installed apps, Apex Booster+ may list launchable apps available on the device.

  • No unrestricted access: the app does not request permission for unrestricted access to the full system app list.
  • No external transmission: the list of installed apps is never sent to servers.
  • User control: the user decides which apps to add to the library. The app does not perform automatic scans or export this data.

6. Data Not Collected

Apex Booster+ does not collect any of the following data:

  • Name, e-mail, or any personal identifier of the user.
  • Precise or approximate location.
  • Contacts, photos, videos, or personal files.
  • Messages, calls, or communication data.

7. Data Sharing

Apex Booster+ does not sell, rent, or share personal data with advertisers, partners, or third parties. There is no:

  • Advertising SDK.
  • Third-party analytics or attribution SDK.
  • Behavioral tracking for advertising purposes.
  • Cookies or tracking technologies.

8. Permissions Used

PermissionPurpose
InternetBasic latency and connectivity test (Apex Ping)
Focus Mode / Do Not Disturb accessGamer Focus Mode — only when the user grants permission

No additional sensitive permissions are requested in this version.

9. Limitations and Important Notices

  • Apex Booster+ does not automatically modify third-party games.
  • The app does not promise actual improvement in FPS, GPU, resolution, ping, or performance of external games.
  • The displayed metrics (RAM, Apex latency) are local and informational — they do not represent automatic changes to the device or to the games.
  • Gamer Focus Mode enables Android's Do Not Disturb only when the user explicitly grants permission.
